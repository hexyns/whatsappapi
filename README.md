# WAG 
* `WAG is a Simple shell script to generate WhatsApp apis`

## Usage
Clone the Repo
```bash
git clone https://github.com/hexyns/whatsappapi
```
Go to the WAG Directory
```bash
cd whatsappapi
```
Run the Script !
```bash
bash WAG
```

## Note
You can enter the phone number in both form

```bash
910123456789
```
```bash
+910123456789
```

* `where 91 or +91 is the country code `
